# 2. 项目背景

企业当前测试流程存在以下问题：

- 手工测试成本高
- 回归测试耗时长
- 自动化脚本分散
- Jenkins 流水线不可视化
- 缺少统一报告中心
- 缺少质量趋势分析
- 测试数据混乱
- 缺少 AI 辅助能力

因此需要建设统一的：

企业自动化测试管理平台。
